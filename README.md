# SolidStats

Десктопное приложение со статистикой проекта **SolidGames · Arma 3**.

## Запуск

Нужен [Node.js](https://nodejs.org/) версии 18 или новее.

```bash
# 1. Установить зависимости (только при первом запуске)
npm install

# 2. Запустить приложение
npm start
```

## Сборка установщика для Windows

```bash
npm run build
```

Готовый `.exe` появится в папке `dist/`.
